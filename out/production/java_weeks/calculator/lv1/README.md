## 프로그램 목적
사칙연산을 수행하는 계산기 개발
## 기능 목록
- [x] 양의 정수 입력받기
  - [x] [Exception] 양의 정수가 아닌 수가 입력되면 예외가 발생한다.
- [x] 사칙연산("+", "-", "*", "/") 기호 입력받기
  - [x] [Exception] 사칙연산 이외의 문자가 입력되면 예외가 발생한다.
- [x] 입력받은 사칙연산 기호를 통해 입력받은 양의 정수 계산하기
  - [x] [Exception] 나누기 연산일 때 분모로 0이 입력되면 예외가 발생한다.  
- [x] "exit"이 입력될 때까지 위 과정 반복
